# Project 4 – FPGA Popcount (Memory-Mapped I/O)

## Overview
This project implements a popcount (bit-counting) accelerator using FPGA and communicates via AXI4-Lite memory-mapped I/O.

---

## Objective
- Design register-based hardware interface  
- Enable CPU ↔ FPGA communication  
- Implement cumulative popcount  

---

## Implementation
- Designed hardware module with AXI4-Lite interface  
- Used memory-mapped registers for control and data  
- Implemented reset and accumulation logic  

---

## Key Features
- Register-level hardware control  
- Memory-mapped communication  
- Stateful computation  

---

## What I Learned
- MMIO design  
- Register interfaces  
- CPU-FPGA communication
