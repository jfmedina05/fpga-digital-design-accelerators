# Project 2 – Exponential Moving Average (C Acceleration)

## Overview
This project accelerates an Exponential Moving Average (EMA) filter by implementing it in C and integrating it with Python.

---

## Objective
- Improve performance of EMA filtering  
- Integrate C with Python using the Python C API  
- Benchmark performance improvements  

---

## Implementation
- Converted EMA recurrence into bit-shift-friendly form  
- Implemented filter in C  
- Exposed function to Python  

---

## Results
- Achieved ~30× speedup compared to Python implementation  

---

## Key Concepts
- Low-level optimization  
- Python C extensions  
- Digital signal processing  

---

## What I Learned
- How to optimize algorithms using C  
- Interfacing Python with compiled code  
- Performance profiling and benchmarking
