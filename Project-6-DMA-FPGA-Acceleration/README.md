# Project 6 – DMA-Based FPGA Acceleration

## Overview
This project introduces DMA to accelerate data transfer between CPU and FPGA for popcount computation.

---

## Objective
- Reduce CPU overhead  
- Enable high-throughput data transfer  
- Integrate DMA with FPGA design  

---

## Implementation
- Used AXI DMA for block data transfer  
- Modified hardware to support streaming input  
- Integrated Python DMA interface  

---

## Results
- Reduced runtime from ~50 seconds to <1 second  

---

## Key Concepts
- DMA (Direct Memory Access)  
- AXI-Stream integration  
- High-throughput computing  

---

## What I Learned
- Designing for throughput vs latency  
- Efficient data movement  
- System-level performance optimization
