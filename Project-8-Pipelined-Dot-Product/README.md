# Project 8 – Pipelined Dot Product

## Overview
This project optimizes a dot product implementation using pipelining to increase hardware throughput.

---

## Objective
- Improve computation speed using pipelining  
- Reduce execution cycles  
- Optimize hardware datapath  

---

## Implementation
- Modified dot product hardware design  
- Inserted pipeline stages  
- Used FMAC (floating multiply-accumulate) units  

---

## Results
- Increased throughput significantly  
- Improved performance over sequential design  

---

## Key Concepts
- Pipelining  
- Throughput vs latency  
- Hardware optimization  

---

## What I Learned
- Designing pipelined architectures  
- Managing data dependencies  
- Performance tradeoffs in hardware
