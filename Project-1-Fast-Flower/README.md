# Project 1 – Fast Flower Generator

## Overview
This project generates parametric flower patterns using polar coordinate transformations. It focuses on efficient computation of geometric shapes and visualization-ready point generation.

---

## Objective
- Generate complex flower patterns using mathematical functions  
- Convert polar coordinates to Cartesian coordinates  
- Efficiently compute and store large sets of points  

---

## Implementation
- Used polar equation:
  - r = a * cos(5θ) + offset
- Converted to Cartesian coordinates for plotting  
- Generated smooth curves using high-resolution sampling  

---

## Key Features
- High-resolution parametric curve generation  
- Efficient point computation and sorting  
- Support for multiple flower instances  

---

## What I Learned
- Polar-to-Cartesian transformations  
- Efficient numerical computation  
- Structuring reusable geometry functions
