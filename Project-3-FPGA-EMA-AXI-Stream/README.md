# Project 3 – FPGA EMA Filter (AXI Stream)

## Overview
This project implements an EMA filter in FPGA hardware using SystemVerilog and the AXI-Stream protocol.

---

## Objective
- Offload EMA computation to FPGA  
- Implement streaming data processing  
- Integrate Python with hardware  

---

## Implementation
- Designed EMA module in SystemVerilog  
- Used AXI-Stream interface (TVALID, TREADY)  
- Simulated and deployed using Pynq  

---

## Key Features
- Streaming hardware computation  
- Stateful filter implementation  
- Real-time data processing  

---

## What I Learned
- AXI-Stream protocol  
- Hardware/software co-design  
- FPGA development workflow
